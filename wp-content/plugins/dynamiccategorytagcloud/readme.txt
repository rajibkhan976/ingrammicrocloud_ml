Displays the tag cloud dynamically from related articles belonging to the category of the display article

== Description ==

You are using a tag cloud of the standard, and you do not want to display tags that belong to the category not related to the article.
Try this widget.
Based on the category of the display article, this widget gets the tag from the article categories of related range.
For the article there is no category of post page, etc., or by setting `all category`, `internal link article`, it displays the tag by examining the related category.
If you check the 'internal link article', get a category from the internal links to articles.

Select Category related range

 * All Category (unrelated article)
 * Current article Category
 * Include child category tree
 * Include parent category tree
 * Include internal linked article Category
 * Exclude Category

Other parameters : Display format, Sort by, Order, Font size, Number of tags to display, ...

[日本語の説明](http://celtislab.net/wp_plugin_DCTagCloud/ "Documentation in Japanese")

== Installation ==

1. Upload the `DynamicCategoryTagCloud` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new widget called `Dynamic Tag cloud` of the widget page is added
4. Drag and drop the sidebar "dynamic tag cloud" widget, set the parameters
